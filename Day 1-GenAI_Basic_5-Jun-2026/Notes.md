## 𝗪𝗵𝗮𝘁 𝗶𝘀 𝗥𝗔𝗚?

RAG stands for:

```text
Retrieval Augmented Generation
```

RAG enables an LLM to use external documents while answering questions.

### Why RAG?

Problem:

```text
LLMs can hallucinate.
```

Solution:

```text
Provide relevant documents before generating an answer.
```

Benefits:

* Better accuracy
* Reduced hallucinations
* Access to enterprise knowledge

## 𝗜𝗺𝗽𝗼𝗿𝘁𝗮𝗻𝘁 𝗜𝗻𝘁𝗲𝗿𝘃𝗶𝗲𝘄 𝗤𝘂𝗲𝘀𝘁𝗶𝗼𝗻𝘀

### What is GenAI?

AI that generates new content such as text, images, code, audio, and video.

### What is RAG?

A technique that retrieves relevant documents before sending context to an LLM.

### What is an Embedding?

A numerical representation of text used for semantic search.

### What is a Vector Database?

A database optimized for storing and searching embeddings.

### Difference Between GenAI and Agentic AI?

GenAI generates content while Agentic AI can plan, use tools, and execute tasks.
