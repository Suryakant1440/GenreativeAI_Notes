A Vector Database is a specialized database designed to store, index, and search vector embeddings generated by embedding models.

Unlike traditional databases that search using exact matches or SQL queries, vector databases perform semantic similarity searches, allowing AI systems to find information based on meaning rather than keywords.

Simple Analogy
Traditional Database
       ↓
Search by Keywords

Vector Database
       ↓
Search by Meaning
Example

User asks:

How does Azure manage Kubernetes?

Vector Search retrieves:

AKS is Microsoft's managed Kubernetes service.

Even though the exact words are different.
